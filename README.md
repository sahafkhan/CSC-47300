# SK Restaurant

Luxury fine-dining restaurant website project.

## Features
- Responsive navigation with hamburger menu
- Hero section
- Menu cards
- Gallery image slider
- About section
- Contact form and Google Map
- Footer with business hours and social links
- JavaScript shopping cart with quantity, remove, total, and clear cart
